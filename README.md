# AWS Adaptive MFA System

An adaptive, risk-based multifactor authentication system built on AWS that combines multiple authentication factors with intelligent risk detection.

## Project Overview

This system implements:
- Multiple authentication factors (password, OTP, biometric)
- Risk-based authentication that adapts security requirements based on user behavior
- Continuous session monitoring
- Tamper-proof blockchain-style logging using AWS QLDB
- Real-time anomaly detection
